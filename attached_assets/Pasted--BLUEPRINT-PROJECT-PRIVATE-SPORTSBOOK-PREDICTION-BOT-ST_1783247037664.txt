# BLUEPRINT PROJECT: PRIVATE SPORTSBOOK PREDICTION BOT (STAKE)

## I. DAFTAR KEBUTUHAN (DAPUR REKAYASA & DATA)
[ ] Bahasa Pemrograman: Python (Sangat direkomendasikan karena ekosistem data & bot paling matang).
[ ] Akun Penyedia Data Olahraga Historis (Gratis):
    - Football-Data.co.uk (Untuk file CSV sepak bola historis lengkap).
    - FBref / Soccerdata library (Untuk statistik modern seperti xG jika dibutuhkan).
[ ] Akun API Odds & Jadwal (Gratis):
    - The Odds API (the-odds-api.com) -> Untuk dapetin pasaran odds real-time (500 request/bulan gratis).
    - API-SPORTS (api-sports.io) -> Opsi cadangan untuk jadwal real-time (100 request/hari gratis).
[ ] Framework Pengambil Data Stake (Scraper):
    - Python Requests / HTTPX (Untuk tembak API internal Stake).
    - Playwright atau Selenium (Sebagai cadangan jika proteksi Cloudflare Stake sedang ketat).
[ ] Bot Telegram / Discord (Gratis):
    - Telegram Bot API & Library `python-telegram-bot` (Untuk kirim sinyal langsung ke HP pribadi).
[ ] Server / Tempat Menjalankan Skrip:
    - Komputer/Laptop Pribadi (Fase pengembangan).
    - VPS Murah / Cloud Gratisan (Misal: PythonAnywhere atau Render) -> Agar skrip bisa jalan 24/7 otomatis tanpa laptop harus menyala.

## II. FITUR UTAMA YANG HARUS ADA (CORE FEATURES)
[ ] 1. Modul "Data Harvester" (Pengambil Data Otomatis)
    - Skrip otomatis mengambil jadwal pertandingan hari ini.
    - Skrip otomatis mengambil odds terbaru dari pertandingan tersebut langsung dari Stake (lewat scraper/API internal).
[ ] 2. Modul "Math Engine" (Kalkulator AI Pribadi)
    - Membaca data historis tim yang bertanding (diambil dari database lokal/CSV).
    - Menghitung probabilitas (peluang % menang) murni dari kedua tim secara matematis bebas emosi.
[ ] 3. Modul "Value Bet Detector" (Pencari Celah)
    - Sistem otomatis mengubah odds Stake menjadi "Peluang Tersirat" dengan rumus: (1 / Odds) x 100.
    - Sistem membandingkan: Jika [Peluang AI Kamu] > [Peluang Tersirat Stake], maka sistem menandainya sebagai "VALUE BET".
[ ] 4. Modul "Bankroll Management" (Pengaman Uang)
    - Skrip menghitung berapa nominal uang yang harus kamu pasang secara proporsional (Misal menggunakan metode Kelly Criterion atau Flat Betting 2% dari modal). Menghindari boncos akibat emosi.
[ ] 5. Modul "Telegram Alert" (Pengirim Sinyal Manual)
    - Mengirim pesan instan ke HP kamu jika dan hanya jika ditemukan taruhan yang menguntungkan (Value Bet).
    - Pesan wajib berisi: Nama Tim (diwakili kode A vs B jika tidak hafal), Prediksi, Odds di Stake saat ini, Rekomendasi besaran modal, dan Link langsung ke pertandingan di Stake.

---

## III. FUNGSI DAN TUGAS AI (MURNI PROBABILITAS ANGKA)
Karena kamu tidak mengikuti olahraganya, AI di sini bertindak sebagai "Analis Dingin Berbasis Angka" dengan fungsi spesifik:
1. Menghilangkan Bias Emosi: AI tidak peduli sejarah atau nama besar tim. AI hanya melihat performa 5-10 pertandingan terakhir (jumlah gol, kebobolan, tren kandang/tandang).
2. Menghitung Probabilitas Murni: Menggunakan algoritma statistik (seperti Poisson Distribution atau Regresi Logistik sederhana via library Scikit-Learn) untuk memprediksi skor atau peluang menang dalam bentuk persentase bersih (Contoh: Tim A punya peluang menang 62%).
3. Koreksi Data Eksternal (Opsional): Jika kamu integrasikan dengan API AI Generatif (seperti OpenAI/Gemini), fungsinya hanya untuk melakukan "Web Scraping Berita" secara kilat untuk mendeteksi jika ada info cedera mendadak (mengurangi poin prediksi jika pemain kunci absen).

## IV. ALUR KERJA SISTEM (WORKFLOW HARI-HARI)
1. Jam 08:00 Pagi -> Skrip otomatis bangun, menarik jadwal pertandingan hari ini & menarik odds dari Stake.
2. Jam 08:05 Pagi -> AI memproses semua data pertandingan tersebut menggunakan database historis.
3. Jam 08:10 Pagi -> Sistem memfilter mana pertandingan yang "Salah Harga" (Value Bet).
4. Jam 08:11 Pagi -> Bot Telegram berbunyi di HP-mu memberikan rekomendasi.
5. Selesai -> Kamu buka HP, klik link Stake yang dikirim bot, pasang manual dalam 5 detik, lalu lanjut beraktivitas.
